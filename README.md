# ABLY Front-end  과제

- A + B + C 스펙 모두 구현

## 사용 기술

- Editor/IDE: Intellij
- Language: TypeScript
- Framework: React

## 구현 범위
#### 비밀번호 재설정 페이지 ( as A )
1. 인증 코드 발급 요청 페이지
2. 인증 코드 검증 페이지
3. 비밀번호 변경 페이지

### 실행
``` 
npm install // 필요한 dependency들을 설치합니다.
```

``` 
npm start
``` 

Open [http://localhost:3000](http://localhost:3000) to view it in the browser. 🚀
